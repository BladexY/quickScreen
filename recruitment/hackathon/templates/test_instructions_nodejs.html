(epoch time is just a simple way computers use to represent current time as a single float variable. It is the number of seconds that have passed since 1st January, 1970).

You can print the current epoch time in javascript by writing the following code:

console.log(Date.now()/1000)

# Output of the above would be
# 1473636383.373
# Before the fullstop is number of seconds and after the full stop is number of miliseconds

The hash to be generated is as follows:
First calculate the sum of all individual digits before the fullstop as sum1. Then calculate the sum of all individual digits after the fullstop as sum2.

Your hash final hash output will be:
(the current epoch time)-sum1-sum2.
That is, your generated hash output should look something like this:

1473636383.373-44-13
